# 機能開発ロードマップ

## 目的

Fの中心価値である「大量のLeica M262/Q3写真から、残す1枚を最速で選ぶ」を強化する。

以下の既存方針は維持する。

- RAW現像機能は持たない
- 元のDNGには書き込まない
- レート、除外、カラーラベル、キーワードはXMPサイドカーへ保存する
- 写真送りと表示レスポンスを最優先する
- Lightroom / Capture Oneへ安全に引き継げる状態を保つ

## 実装順

### 1. チートシート

新しいショートカットを増やす前に、操作を画面上で確認できるようにする。

- 画面下部に、現在の画面で利用できる主要ショートカットを常時表示する
- `?`で全ショートカットのオーバーレイを開閉する
- オーバーレイは`?`または`Esc`で閉じる
- 下部バーは設定から非表示にできる
- 下部バーの初期値は有効とする
- 表示内容は画面の状態に応じて切り替える

表示例：

```text
←→ 移動   1–5 レート   X 除外   Z 等倍   G グリッド   C 比較   ? ヘルプ
```

画面別の表示候補：

- グリッド：移動、開く、複数選択、評価
- 1枚表示：移動、評価、除外、等倍、グリッド
- 比較表示：候補変更、勝者選択、同期等倍
- 複数選択中：一括評価、一括除外、キーワード

### 2. 未評価フィルターと選別進捗

- 「未評価のみ」「評価済みのみ」のフィルターを追加する
- `選別済み 428 / 1200`のように進捗を表示する
- 「次の未評価へ移動」操作を追加する
- DNG/JPGペアは1ショットとして進捗を計算する
- XMPから読み込んだ既存評価も進捗へ反映する

### 3. Undo / Redo

- `⌘Z`で直前のメタデータ変更を取り消す
- `⇧⌘Z`で取り消した変更をやり直す
- 対象はレート、除外、カラーラベル、キーワードとする
- DNG/JPGペアへの変更は1操作として扱う
- 一括操作も1回のUndoで元に戻せる履歴構造にする
- XMP書き込みの直列化とrevision管理を維持する

### 4. 評価後の自動送り

- レート、除外、カラーラベルを設定した直後に次の写真へ移動する
- 設定から有効・無効を切り替えられるようにする
- 既存の操作感を変えないため、初期値は無効とする
- Undo時は変更対象だった写真へ戻る
- 最後の写真では現在位置を維持する

### 5. グリッドの複数選択と一括評価

- `Command`クリックで個別に選択を追加・解除する
- `Shift`クリックで範囲選択する
- レート、除外、カラーラベル、キーワードを一括適用する
- 一括操作全体を1回のUndoで取り消せるようにする
- DNG/JPGペアのメタデータ共有を維持する
- フィルター変更後も可能な範囲で選択状態を維持する

### 6. A/B比較ビュー

- 2枚を左右に表示する
- ズーム倍率とパン位置を同期できるようにする
- ワンキーで勝者を選択する
- 勝者を残したまま次の候補と比較できるようにする
- 100%等倍とフル解像度の遅延デコードを利用する
- 将来的に4枚比較へ拡張できる状態管理にする

### 7. 近接カットの自動スタック（10-A）

10-Bとは独立したオプション機能として実装する。

- 設定の初期値は無効とする
- 初期版は撮影時刻とファイル連番で近接カットをグループ化する
- 次の段階でperceptual hashを追加し、画像の類似度でグループを補正する
- グリッドには代表写真とスタック内枚数を表示する
- スタックは任意に展開・折りたたみできるようにする
- スタック内の写真をA/B比較へ直接送れるようにする
- 解析結果はアプリ用キャッシュへ保存し、DNG/XMPには記録しない

### 8. 技術品質解析（10-B）

10-Aとは独立したオプション機能として実装する。

- 設定の初期値は無効とする
- 既存のサムネイルを利用してバックグラウンド解析する
- 初期版では大規模なAIモデルを使用しない
- 以下の指標を解析する
  - シャープネス
  - 白飛び
  - 黒つぶれ
  - コントラスト
- 総合点だけでなく、判定根拠を個別に表示する
- 10-Aが有効な場合は、スタック内の相対順位も表示する
- 10-Aが無効な場合も、個々の写真に対する技術的警告として動作する
- 写真送りを最優先し、連続操作中は解析を一時停止または抑制する
- 解析結果はファイルサイズと更新日時を含む指紋とともにキャッシュする
- 新規または更新されたファイルだけを再解析する
- 将来的にmacOSのVisionによる顔検出を任意機能として追加できる構造にする

## 設定項目

```text
操作
  [ ] 評価後に自動で次へ進む

表示
  [✓] 下部にショートカットを表示

選別アシスト
  [ ] 近接カットを自動スタック
  [ ] 技術品質をバックグラウンド解析
```

## 依存関係

- チートシートは、後続機能で増えるショートカットの案内基盤とする
- Undo / Redoを一括評価より先に実装し、誤操作を安全に取り消せるようにする
- A/B比較ビューは既存のMetal描画、テクスチャキャッシュ、等倍表示を再利用する
- 10-Aと10-Bはそれぞれ単独で有効化できるようにする
- 10-Aと10-Bを同時に有効化した場合のみ、スタック内品質ランキングを追加表示する

## 対象外

このロードマップでは、以下を対象外とする。

- RAW現像や画像補正
- DNG本体へのメタデータ書き込み
- クラウドへの写真アップロード
- ユーザーの作風を学習する個人化モデル
- AIによる自動除外や自動レート確定

品質解析はあくまで判断材料を提示する機能とし、最終的な選択はユーザーが行う。
